# Project Portfolio 2

A simple HTML/CSS/JavaScript portfolio project showcasing basic web development skills.

## 📌 Overview

This repository contains a personal portfolio page built with:
- **HTML** (`main (3).html`) – structure of the site  
- **CSS** (`style (3).css`) – styling and layout  
- **JavaScript** (`script.js`) – interactivity and dynamic behavior  

The project demonstrates core front‑end techniques and serves as a foundation for expanding into more complex web apps.

## 🔧 Features

- Responsive layout
- Basic navigation and sections (e.g. About, Projects, Contact)
- Interactive elements controlled via JavaScript
- Clean, minimal design suitable for customization

## 🚀 Getting Started

1. **Clone the repository**  
   ```sh
   git clone <your-repo-url>
   ```
2. **Open `main (3).html`** in your browser.  
   No build step or server is required – it’s a static site.

## 🛠️ Customization

- Modify `style (3).css` to change colors, typography, or layout.
- Add new sections or content directly in `main (3).html`.
- Enhance interactivity by editing `script.js`.

## 📁 Project Structure

```
main (3).html      – main HTML file
script.js          – JavaScript for page behavior
style (3).css      – stylesheet
README.md          – this file
```
